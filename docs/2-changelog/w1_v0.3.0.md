# Changelog - Week 1, 30-04-2026, V. 0.3.0

**Release Date**: Week 1, 30-04-2026 at 16:52
**Version**: 0.3.0 (previously 0.2.0)
**Object**: Add Phase B3 UI + authoring: XCAF migration, PySide6 app, panels, overlays, run dispatch
**Code review**: `docs/3-code-review/CR_w1_v0.3.0.md` (Codex loop, 9 rounds → NEEDS REVISION with open plan-conformance items)

## Changes

- **XCAF Migration (B3.0)**: Migrated `step_loader` from `STEPControl_Reader` to
  `STEPCAFControl_Reader`. `LeafSolid` gains `name`, `color_rgb`, `material_hint`
  fields from XCAF. `AssemblyNode` gains `name`. `extract_step_tags()` simplified
  to a pure mapping from `LeafSolid` fields — no longer reads the STEP file a
  second time.

- **PySide6 Desktop Application (B3.1–B3.7)**: Added `raysim gui` CLI subcommand
  and the full `raysim.ui` package:
  - `app.py`: QMainWindow with dockable panels, QSettings persistence, menus
  - `viewer.py`: OCCT AIS 3D viewer with face-centroid picking, snap modes,
    normal-offset placement, bidirectional selection sync
  - `panels/tree_panel.py`: Assembly tree with XCAF colors and material status
  - `panels/material_panel.py`: Library browser, auto-assignment status, accept-all
  - `panels/detector_panel.py`: Point placement with snap modes and face-normal offset
  - `panels/scenario_panel.py`: .dos picker, DDC matplotlib preview, Nside spinbox
  - `panels/run_panel.py`: Run button with gating, progress bar, cancel, save dialog
  - `panels/result_panel.py`: Per-detector results table with overlay checkboxes
  - `workers/run_worker.py`: QThread dispatch with max-hit fatal gate, provenance
  - `overlays/ray_overlay.py`: 3D ray-view colored by mm-Al
  - `overlays/mollweide_overlay.py`: Matplotlib Mollweide angular shielding
  - `overlays/projection_overlay.py`: 6-face unfolded box projection

- **State Controller**: `AppState` owns all mutable state, manages project lifecycle,
  dirty tracking, scene rebuild on assignment changes, and run-readiness gating.

- **Tests**: Updated `test_step_loader.py` with XCAF field and DFS regression tests.
  Updated `test_mat_step_tags.py` for simplified signature. Added `test_run_worker.py`
  for RunContext validation.

- **ARCHI.md**: Updated §9b/§9c to document B3.0 XCAF migration and simplified
  `extract_step_tags(leaves)` API.
